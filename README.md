# UI_BAi — Color Intelligence Tool

A professional color tool for designers and developers.

## Features
- Compare two colors side by side
- Convert between HEX, RGB, HSL, HSV, CMYK
- WCAG contrast checker (AA/AAA)
- Color harmony palette generator
- CSS Gradient builder
